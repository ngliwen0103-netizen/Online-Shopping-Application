# RELM Gaming Gear E-Commerce Mobile Application
## Features
- Secure login and user account management
- Product catalogue with category search engine
- Product information, specifications and ratings
- Recommendation system & best seller
- Handling shopping cart
- Order placement & checkout process
- Ability to review and rate products
- Keep track of purchase history
- User-friendly and responsive mobile interface
- Persistence of local data
